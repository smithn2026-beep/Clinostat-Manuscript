# Clinostat Manuscript
Clinostat Manuscript repository

GFP folder contains fluorescent microscope images for PIN1::GFP and DR5::GFP seedlings. DR5:GFP seedling folders contain additional screenshots of the 8-but image with histogram analysis.

Genomic data contains the eggNOG mapper results for _Sphingomonas sanguinis_ B3A genome. Supplamental figure contains in depth analysis using KEGG pathway tool BlastKOALA to identify important (and present) pathways identified from eggNOG mapper. 80+ genes that are involved in plant-microbe interactions and spaceflight survival were identified and noted.

Root curvature contains results pertaining to clinostat method verification using Clary et al. methods.

Mass + Root Hair folder holds data collected from phenotyping experiments conducted by inoculating _Arabidopsis thaliana_ with _S. sanguinis_ B3A.
